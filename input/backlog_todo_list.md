# Todo List Backlog - SARI Project

## Epic: Todo List Application
- Type: Epic
- Description: Full-stack Todo List application with Angular frontend, state management, REST API backend, and end-to-end test coverage. Create this epic first, then link all stories below as children.

### Stories (link each story to the epic above as parent)

- **Story 1: Todo List Angular UI Components**
  - Type: Story
  - Description: Create Angular components for displaying, adding, editing and deleting todo items with a responsive layout.

- **Story 2: Todo List State Management Service**
  - Type: Story
  - Description: Implement an Angular service with in-memory state management for todo items using RxJS observables.

- **Story 3: Todo List REST API with Express**
  - Type: Story
  - Description: Build a Node.js/Express REST API with CRUD endpoints for todo items and MongoDB persistence.

- **Story 4: Todo List API Integration and E2E Tests**
  - Type: Story
  - Description: Connect the Angular frontend to the REST API and write end-to-end tests with Cypress.
